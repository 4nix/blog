一. 键值为数组时的情况

{% highlight js %}
console.log(["a","b","c","d","e"][[1,2],3,4]);
{% endhighlight %}

这个结果是 "e", 是的, 你没看错是 "e"!

这是因为在js中, []后面跟的应该是键值, 就是 [数组][键值], 而键值是整数, 所以js把 "[1,2],3,4"转成了整数!!!,

再试试 console.log(([1,2],3,4)) 结果就是4, 根据类型转换的原则, 这里是取了 ([1, 2], 3, 4)的原始值, 也就是(...).valueOf()了
