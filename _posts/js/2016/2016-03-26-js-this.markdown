1)  当在全局范围使用, this 指向全局
2) 在函数内调用, foo(),  this 指向全局
3) 方法中调用, test.foo(), this 指向test
4) 调用构造函数, new foo(), this 指向新创建的对象
5) 显示设置this, 比如使用 apply, call, 指向第一个参数
6) 闭包中的函数会指向全局,
    比如 Foo.method = function() {
         test() {
              // 这里使用this 会指向全局
         }
         test();
      }
